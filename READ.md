# Freelance Finance Deep Learning Project

This project investigates whether deep learning models can predict financial behavior patterns from personal finance transaction data.

## Dataset

This project uses the BudgetWise Personal Finance Dataset. The dataset contains transaction-level financial data including user ID, date, transaction type, category, amount, payment mode, location, and notes.

A copy of the dataset is included in the `data/` folder for this milestone.
